# 👥 User Personas — Artisan Market

---

## 1. Fatou – The Artisan

| | |
|---|---|
| **Age** | 34 |
| **Occupation** | Handmade jewelry creator |
| **Location** | Dakar, Senegal |
| **Problem** | No physical store, hard to find customers |
| **Goal** | Show her products online easily and get more sales |
| **Tech Level** | Intermediate — comfortable with basic apps |
| **Solution** | Post digital flyers with photos, prices, and contact info |

**Quote:** *"I make beautiful jewelry, but people don't know where to find me."*
